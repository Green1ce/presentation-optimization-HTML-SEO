Internal optimization

  • < tags > - wireframe of page
  • 'attributes' - extend or modify tags

Tags

  • paired (< body > < /body >)
  • single (< img >)

Three main HTML tags:

  • < html > - marks the boundaries of the document;
  • < head > - contain information for browser and search engines;
  • < body > - visible part of the document;

< head > includes:

  • < title > < /title >
  • < meta >
  • < link >

< title >

  • naming the document
  • title of the page
  • name on browser window
  • link in bookmark

< meta >

  • name = 'description';
  • name = 'robots';
  • name = 'charset';
  • name = 'viewport';
  • name = 'keywords'.

description

A short description of the page content

robots

The rules for indexing specific pages of the site

and follow the hyperlinks from these pages.

Main values 'content' attribute:

  • content = 'index';
  • content = 'noindex';
  • content = 'follow';
  • content = 'nofollow';

charset

The type of encoding of the document

viewport

Processing page sizes for the user's device

keywords

Contains the keywords of the page content

< link >

Links the document to other web pages

alternate

Used to indicate alternative versions of site pages

Website content optimization

Need to use tags correctly

  • < div > - used to create a wireframe of the page;
  • < span > - used for horizontally positioned pieces of content;
  • < table > - used to create tables;

Website content optimization

Need to use tags correctly

  • < h1 > ... < h6 > - used for headings;
  • < p > - used to divide text into paragraphs;
  • List design tags:
  • < ul > - to determine the items to be listed;
  • < li > - for bulleted listed;
  • < ol > - for numbered listed;

Website content optimization

Need to use tags correctly

< img >

  • 'src' - defines the name and position of the image;
  • 'alt' - alternative textual description of the image;
  • 'title' - pop-up image description on hover;

HTML5

New optimization tags

  • < header > - informs about the content of the page;
  • < nav > - placement of the main site navigation;
  • < article > - description of the site's article;
  • < aside > - contains part of the content of the site article;
  • < section > - divides the main content into subtopics;
